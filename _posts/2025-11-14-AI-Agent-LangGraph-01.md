`AI Agent`란 AI를 통해 의사결정 및 판단하는 프로그램이며 `LLM Agent`는 Agent 중 하나로 활용되는 AI가 LLM이다. 좋은 Agent를 만들려면 LLM을 잘 활용할 수 있어야 한다.

그러면 어떻게 LLM을 잘 활용하여 Agent를 쉽게 개발할 수 있을까? 

이를 위해서는 `LangGraph`라는 도구를 사용할 수 있다.

`LangGraph`는 다음과 같은 특징이 있다.

- Statefule
- Orchestration
- LangChain 문법을 기반으로한 framework


## LangGraph 기본 문법
--------

LangGraph의 기본 문법으로는 총 3가지가 있다.

1. State
2. Node
3. Edge

## LangGraph 응용 방법
------

1. Tool Calling
2. ReAct Agent
3. Custom Agent
