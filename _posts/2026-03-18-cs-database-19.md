## MVCC

---

`MVCC(Multi Version Concurrency Control)`는 Lock의 단점인 적은 트랜잭션의 처리량을 보완하기 위해서 고안된 기술이다.

|       | read | write |
| :---: | :--: | :---: |
| read  |  O   |   O   |
| write |  O   |   X   |
